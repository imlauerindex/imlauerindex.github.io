Luego de la instalación para bootear FreeMCBoot tenés que encender la PS2 sin el CD y darle a Browser seleccionar la memory card en donde instalaste funtuna y dale redondo redondo. Los MP4 solo los reproduce como audio usando Simple Media System(SMS) no soporta videos en HD , sólo probé con videos de youtube en 360 con el siguiente comando: yt-dlp -f 134+140 video_url y luego lo transformé a avi usando este comando: ffmpeg -i input.mkv -map 0:v:0 -map 0:a:0 -c:v mpeg4 -vtag xvid -b:v 3000k -vf scale=880:-1 -ac 2 -c:a libmp3lame -b:a 224k -af "aresample=matrix_encoding=dplii" output.avi
